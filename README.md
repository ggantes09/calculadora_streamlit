# Minha Calculadora

A simple calculator built with Python and Streamlit, with operation history.

## What it does

- Performs the 4 basic mathematical operations (addition, subtraction, multiplication and division)
- Keeps a history of all operations performed in the session
- Handles division by zero with a friendly error message

## How to use

1. Clone the repository
2. Install the dependencies:
pip install streamlit
3. Run the app:
streamlit run app.py
